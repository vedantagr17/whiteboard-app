# Dynamic Whiteboard Application

## Description

This React-based whiteboard application allows users to create and collaborate on interactive drawings in real-time. It features a clean and responsive design built with Tailwind CSS and is deployed on Vercel for optimal performance and accessibility.

## Features

- **Drawing Tools**: Draw lines, rectangles, ellipses, arrows, and add text with various customization options.
- **Undo/Redo**: Easily undo or redo your drawing actions.
- **Real-time Collaboration**: Multiple users can collaborate on the same whiteboard simultaneously.
- **Responsive Design**: Works seamlessly across different devices and screen sizes.
- **Performance Optimization**: Optimized for smooth drawing and rendering, even with complex drawings.
- **Deployment**: Deployed on Vercel for easy accessibility and scalability.

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/vedantagr17/whiteboard-app.git
    ```
2. Install dependencies:
    ```sh
    npm install
    ```
3. Start the development server:
    ```sh
    npm start
    ```

## Usage

1. Open your web browser and navigate to `http://localhost:3000`.
2. Start drawing on the whiteboard using the available tools.
3. Collaborate with other users by sharing the whiteboard link.

## Technologies Used

- React
- HTML5 Canvas
- Tailwind CSS
- Vercel
- Context API
